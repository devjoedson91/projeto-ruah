# Como executar o projeto

Pré-requisitos: Node js, Express, mysql2

```bash
# clonar repositório
git clone https://github.com/devjoedson91/projeto-ruah.git

# entrar na pasta do projeto no cmd ou git bash
cd projeto-ruah

# executar o projeto
set DEBUG=projeto-ruah:* & npm start
```
